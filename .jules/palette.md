
## 2025-03-21 - Retaining Context in CLI Applications
**Learning:** When users clear the screen in a CLI application (e.g., via a `clear` or `cls` command), they often lose the context of the application they are running, leading to a blank, disorienting state.
**Action:** Always re-print the application header, greeting, or essential context after executing a clear screen command to maintain a continuous and guided user experience.
